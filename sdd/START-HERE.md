# START-HERE

Use este arquivo como instruicao curta de bootstrap.

## Leitura minima
1. `sdd/README.md`
2. `sdd/IGNITION-PROMPT-CODEX.md` ou `sdd/IGNITION-PROMPT.md`
3. `sdd/PROJECT-INIT-lite.md`
4. `sdd/PROJECT-INIT.md`
5. `sdd/AGENTS.md`
6. `sdd/WORKFLOW.md`
7. `sdd/ai-manifest.yaml`

## Bootstrap recomendado
1. preencher `PROJECT-INIT-lite.md` ou `PROJECT-INIT.md`
2. usar `IGNITION-PROMPT-CODEX.md` no Codex ou `IGNITION-PROMPT.md` para acionar a IA
3. consolidar contexto em `context/`
4. criar a primeira SPEC em `specs/`
5. definir contratos em `contracts/`
6. registrar ADRs em `decisions/`
7. montar plano em `execution/`
8. derivar testes em `tests/`

## Regra pratica
- nao leia a arvore inteira sem necessidade
- trate `specs/` como fonte de verdade funcional
- trate `contracts/` como fonte de verdade tecnica
- trate `execution/` como estado atual do trabalho
